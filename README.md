# Beauty Shop App

Beauty Shop App is a modern Android e-commerce application developed using Kotlin and Firebase.
The application allows users to explore beauty products, search items, add products to cart, manage favourites, and view product details in a smooth and user-friendly interface.

The app also includes an admin section for managing product-related information through Firebase Realtime Database.

# Features

## User Features

* User Registration and Login using Firebase Authentication
* Email Verification System
* Product Search Functionality
* Product Categories
* Product Details Page
* Add to Cart System
* Favourite Products System
* Product Sharing Option
* Profile Section
* Logout Functionality
* Realtime Product Updates using Firebase Realtime Database

---

## Admin Features

* Separate Admin Access
* Admin Dashboard
* Manage Products
* Manage Product Prices
* Manage Product Quantity
* Firebase Realtime Database Integration

# Firebase Features

This project uses Firebase Authentication for secure login and registration.
Firebase Realtime Database is used to store and update product information dynamically.

Any changes made in Firebase Realtime Database are updated instantly in the application.
